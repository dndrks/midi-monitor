# midi-monitor

A MIDI Monitor script for norns

v0.5 @okyeron


__Requirements:__
  * norns version 200424 (for global clock)
  * midi devices

__Controls:__  

E1 - select MIDI device
E3 - scrollback through MIDI messages

K2 - toggle unmute/mute internal synth playback
K3 - clear screen / buffer
